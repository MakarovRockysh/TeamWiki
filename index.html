设计原则

不绑定单一工具

同一份知识库，可以同时输出 .cursor/rules、AGENTS.md、CLAUDE.md。换工具不会断。

Markdown 是真源

所有内容以 markdown 存在 git，TeamWiki 只是视图层。个人迁移、团队分叉都成立。

写 wiki 是副产物

通过 Agent 自动归档与索引，避免"组织号召大家写"的反人性流程。
