# 2026-05-03 · Sprint 1 主壳搭建过程

## 结论 / 范围

1. **技术选型**：纯静态 HTML + CSS + 原生 JS。理由：零依赖、立刻能跑、Cloudflare Pages 友好、不绑定任何前端框架。
2. **样式 100% 沿用 `idea_share_html`**：液态光斑背景、`.glass` 玻璃材质、Inter + JetBrains Mono、深色主题。
3. **架构落地**：4 层（资产 / 沉淀 / 实践 / 元）× 13 模块全部建成，每个模块都有真实示例数据 + 至少 1 个交互。
4. **共享骨架自动注入**：`scripts/layout.js` 在每个页面自动注入顶部 nav + 侧边 sidebar + 底部 footer，新增页面只需写主体内容。
5. **本地启动**：`python3 -m http.server 8088` —— http://127.0.0.1:8088/
